# Enterprise CRM System

A full-stack CRM (Customer Relationship Management) application built using React, Node.js, Express, and PostgreSQL.

## Features

* User Authentication using JWT
* Role-Based Access Control
* Employee Management
* Opportunity Management
* Dashboard Analytics
* File Upload Support
* Secure API Endpoints

## Tech Stack

### Frontend

* React.js
* Axios
* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* PostgreSQL
* JWT Authentication
* Multer

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
