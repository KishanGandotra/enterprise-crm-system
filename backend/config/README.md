# Enterprise CRM System

A full-stack CRM (Customer Relationship Management) platform built using React, Node.js, Express, PostgreSQL, JWT Authentication, and Multer.

## Features

### Authentication & Authorization

* JWT Authentication
* Role-Based Access Control
* Admin Dashboard
* Employee Dashboard

### Employee Management

* Create Employee
* Update Employee
* Delete Employee
* View Employees

### Opportunity Management

* Create Opportunities
* Edit Opportunities
* Delete Opportunities
* Assign Opportunities
* Update Opportunity Status

### Collaboration

* Opportunity Notes
* Activity Logs
* Opportunity History Tracking

### Analytics Dashboard

* Total Revenue
* Pipeline Revenue
* Won Deals
* Lost Deals

### Productivity Features

* Search Opportunities
* Status Filtering
* Pagination

### File Management

* Upload Documents
* View Attached Files
* Download Files

## Tech Stack

### Frontend

* React.js
* Axios

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL

### Authentication

* JWT

### File Storage

* Multer
